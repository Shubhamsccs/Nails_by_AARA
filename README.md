# Nails by AARA ✦

A responsive, elegant, and interactive single-page portfolio website designed for a luxury minimalist nail artist based in Mumbai, India. The project features a refined aesthetic, smooth animations, and a seamless user experience tailored for editorial-standard nail care services.

## ✨ Features

* **Custom Interactive Cursor**: A stylized dot-and-ring custom cursor that dynamically expands and changes color when hovering over interactive elements.
* **Smooth Scroll & Reveal Animations**: Utilizes `IntersectionObserver` to reveal elements smoothly as the user scrolls down the page, enhancing the storytelling flow.
* **Parallax & Sticky Navigation**: Features a dynamic scroll progress bar, a sticky navigation menu that appears on scroll, and subtle parallax background text effects.
* **Responsive Layout**: Built with modern CSS Grid and Flexbox, ensuring a flawless layout across desktop, tablet, and mobile devices.
* **Interactive Booking Form**: A fully styled contact/booking form with built-in JavaScript validation and simulated success feedback upon submission.
* **Continuous Marquee**: A CSS-animated marquee strip highlighting key services.

## 🛠️ Technologies Used

* **HTML5**: Semantic markup for a clear, accessible structure.
* **CSS3**: Custom variables, Grid/Flexbox layouts, hover states, and keyframe animations.
* **Vanilla JavaScript**: DOM manipulation, event listeners, custom cursor logic, and intersection observers for scroll animations.
* **Google Fonts**: 
    * *Cormorant Garamond* (Headings, editorial accents)
    * *DM Sans* (Body copy, buttons, UI elements)
